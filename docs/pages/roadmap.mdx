# Hyperscan Roadmap

Our roadmap outlines the key milestones and upcoming features for Hyperscan.

## Q4 2023

- **Enhanced API Security**: Implement OAuth 2.0 for secure API access.
- **New SDKs**: Release SDKs for Ruby and PHP.
- **Performance Improvements**: Optimize AI model inference times.

## Q1 2024

- **Custom AI Models**: Allow users to train and deploy custom models.
- **Expanded Documentation**: Add more examples and tutorials.
- **Partner Portal**: Launch a dedicated portal for partners.

## Q2 2024

- **Multi-Language Support**: Add support for additional languages in the UI.
- **Advanced Analytics**: Provide detailed usage and performance metrics.
- **Beta Features**: Introduce beta access to experimental features.

Stay tuned for updates as we continue to innovate and improve Hyperscan.
