# Cargo Weight Estimation

Estimate cargo weight using advanced AI models and integrate seamlessly with IoT devices. Hyperscan's cargo weight estimation API provides accurate predictions to optimize logistics operations.

## Features

- **AI-Powered Estimation**: Predict cargo weight with high accuracy using machine learning models.
- **Sensor Integration**: Combine AI models with IoT sensors for precise measurements.
- **Customizable Models**: Adapt models to specific cargo types and configurations.
- **Batch Processing**: Estimate weights for multiple cargo items in a single request.

## Input

The cargo weight estimation API requires the following inputs:

- **Cargo ID**: A unique identifier for the cargo item.
- **Dimensions** (optional): Length, width, and height of the cargo.
- **Material Type** (optional): The type of material (e.g., steel, wood, plastic).
- **Sensor Data** (optional): Data from IoT sensors, such as weight readings or density measurements.
- **Historical Data** (optional): Previous weight records for similar cargo types.

### Example Input (JSON)
```json
{
  "cargoId": "cargo-id-123",
  "dimensions": {
    "length": 2.5,
    "width": 1.2,
    "height": 1.0
  },
  "materialType": "steel",
  "sensorData": {
    "weightReading": 1200,
    "density": 7.85
  },
  "historicalData": [
    { "cargoId": "cargo-id-122", "weight": 1180 },
    { "cargoId": "cargo-id-121", "weight": 1195 }
  ]
}
```

## Output

The API returns an estimated weight along with additional metadata:

- **Estimated Weight**: The predicted weight of the cargo.
- **Confidence Score**: A percentage indicating the confidence level of the estimation.
- **Warnings** (optional): Any issues or anomalies detected during the estimation process.

### Example Output (JSON)
```json
{
  "estimatedWeight": 1205,
  "confidenceScore": 95,
  "warnings": ["Sensor data partially missing"]
}
```

## Supported Features

- **Real-Time Estimation**: Get weight predictions instantly for operational efficiency.
- **Batch Estimation**: Process multiple cargo items in a single API call.
- **Customizable Models**: Train models for specific cargo types or industries.
- **IoT Integration**: Seamlessly integrate with IoT devices for enhanced accuracy.
- **Error Handling**: Detect and handle anomalies in input data.

## Outcomes

- **Improved Accuracy**: Reduce errors in weight estimation, leading to better planning and cost management.
- **Operational Efficiency**: Streamline loading and unloading processes with accurate weight data.
- **Regulatory Compliance**: Ensure compliance with weight restrictions and safety regulations.
- **Cost Optimization**: Avoid overloading vehicles and optimize fuel consumption.

## Disadvantages

- **Dependency on Input Data**: Requires accurate dimensions, material types, or sensor data for optimal results.
- **IoT Integration Complexity**: May require additional setup for IoT device integration.
- **Model Training**: Custom models may need periodic retraining to maintain accuracy.
- **Edge Cases**: Unusual cargo types or configurations may result in lower confidence scores.

## API Examples

### JavaScript
```javascript
client.estimateCargoWeight({
  cargoId: "cargo-id-123",
  dimensions: { length: 2.5, width: 1.2, height: 1.0 },
  materialType: "steel",
  sensorData: { weightReading: 1200, density: 7.85 }
}).then(result => {
  console.log(`Estimated Weight: ${result.estimatedWeight} kg`);
  console.log(`Confidence Score: ${result.confidenceScore}%`);
  if (result.warnings) {
    console.warn(`Warnings: ${result.warnings.join(", ")}`);
  }
});
```

### Python
```python
result = client.estimate_cargo_weight(
    cargo_id="cargo-id-123",
    dimensions={"length": 2.5, "width": 1.2, "height": 1.0},

    sensor_data={"weightReading": 1200, "density": 7.85}

print(f"Estimated Weight: {result['estimatedWeight']} kg")
print(f"Confidence Score: {result['confidenceScore']}%")
if "warnings" in result:
    print(f"Warnings: {', '.join(result['warnings'])}")
```

## SDK Example (Go)
```go
package main

import (
	"fmt"
	"log"
	"hyperscan/sdk"
)

func main() {
	client := sdk.NewClient("your-api-key")

	// Estimate cargo weight
	weightResult, err := client.EstimateCargoWeight(sdk.CargoWeightRequest{
		CargoID: "cargo-id-123",
		Dimensions: sdk.Dimensions{
			Length: 2.5,
			Width:  1.2,
			Height: 1.0,
		},
		MaterialType: "steel",
		SensorData: sdk.SensorData{
			WeightReading: 1200,
			Density:       7.85,
		},
	})
	if err != nil {
		log.Fatalf("Error estimating weight: %v", err)
	}
	fmt.Printf("Estimated Weight: %.2f kg\n", weightResult.EstimatedWeight)
	fmt.Printf("Confidence Score: %d%%\n", weightResult.ConfidenceScore)
	if len(weightResult.Warnings) > 0 {
		fmt.Printf("Warnings: %s\n", weightResult.Warnings)
	}
}
```
